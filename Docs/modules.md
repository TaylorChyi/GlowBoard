模块一：文本内容管理

**模块说明：**负责管理滚动显示的文本内容，满足用户多途径录入和保存文本的需求。用户可以通过手动输入、复制粘贴现有文本，导入本地TXT文稿，或从云端同步（利用iCloud）获取文本。应用支持保存多条文本作为模板，以便日后快速调用。同一账户下的多设备之间可通过iCloud实现脚本的自动同步 ￼。文本内容管理模块还应提供文本列表视图，方便用户新建、选择、重命名或删除保存的稿件。

关键交互流程：
	•	**新建/编辑文本：**用户打开应用后，可点击“新建文本”进入编辑界面，键入或粘贴文字内容。编辑完成后，点击保存，将文本内容存储为一个提词稿模板（字符长度不受限制 ￼）。再次打开时应用会自动加载上次编辑的文本和设置，方便继续使用。
	•	导入本地文件：用户选择“导入文稿”功能，调起系统文件选择器，选定本地的TXT（或兼容的.docx/RTF等格式）文件后，应用读取文件内容并生成新的滚动文本稿件 ￼。导入后用户可对文本进行必要编辑，并保存为模板。
	•	**云端同步：**用户在系统设置中启用iCloud同步后，应用将在每次保存或修改文本时将稿件上传至用户的iCloud云盘或CloudKit容器。其他iOS/iPadOS设备登录同一Apple ID时将自动获取最新稿件列表，实现跨设备无缝衔接 ￼。例如，用户可在Mac上编写稿件，通过iCloud同步后在iPad的GlowBoard中直接使用。
	•	**管理多文本模板：**用户进入“文本库”界面，可浏览已保存的多个稿件条目。支持根据标题或日期排序、搜索关键字等。点击某一条目即可载入该稿件进行滚动显示。用户也可左滑删除不需要的稿件，或复制现有稿件以新建版本。通过AirDrop等分享渠道，用户还能方便地导出或备份稿件 ￼。

模块二：滚动显示与样式设置

模块说明：负责文字滚动呈现和样式调整。GlowBoard提供横向和纵向两种滚动方式，适配不同使用场景（如横向用于LED走马灯显示，纵向用于演讲提词）。用户可调整滚动速度、字体字号、行间距、字体类型，以及文字和背景颜色，以获得舒适的阅读效果 ￼ ￼。应用内置浅色和夜间两套配色方案，并允许用户自定义背景色/文字色搭配。字体设置支持系统已安装字体库，满足不同风格需求 ￼。此外，为减轻长时间阅读的眼部负担，大屏设备上将提供内容区域宽度调节或两侧留边选项，以避免文字一行过长时眼球大量移动 ￼。

关键交互流程：
	•	**切换滚动方向：**用户在设置面板中选择“滚动方式”，可在“纵向卷动”（文字从下向上平滑滚动）与“横向卷动”（文字从右向左连续滚动）之间切换。切换后应用在预览窗口中即时显示新的滚动方向布局，用户确认选择后保存设置。
	•	**调整滚动速度：**在滚动界面，用户通过拖动速度滑杆或点击“+/-”按钮来增减速度，或者直接选择预设的慢/中/快档位。速度调整会实时反映到正在滚动的文本上，实现“所见即所得”的效果 ￼。用户也可设置一个固定时长，由应用自动计算并平滑调整滚动速度以匹配预设的演讲时间。
	•	**修改文字样式：**用户点击样式设置按钮，展开字体和颜色选项面板。通过下拉菜单选择字体，拖动滑块调整字号和行距，颜色选择器切换文字颜色和背景色。所有改动都会即时作用于后台的文本显示区域，方便用户预览效果 ￼。例如，用户将字体从黑体切换为宋体，文本显示随即更新字体；调整行距时，多行文本间距在滚动预览中实时变化。用户满意后关闭面板，设置自动保存。
	•	**夜间模式与亮度：**用户可以一键开启夜间模式，界面整体切换为深色背景、浅色文字的配色方案，同时应用可降低屏幕亮度以适应暗环境（也可通过提供亮度调节滑杆由用户手动控制）。夜间模式下的配色和亮度仅影响显示、不改变文本内容，本质是提供暗环境下更舒适的观感。再次切换回日间模式时，背景和文字颜色恢复默认或之前自定义的配色。

模块三：预览与实时调整界面

**模块说明：**提供滚动效果的预览展示，并允许用户在预览过程中实时调整参数。该模块强调“所见即所得”的交互体验，确保用户在正式演讲/录制前能准确调整到满意的效果。预览界面采用全屏沉浸设计，只在必要时显示极简的控制组件，以免干扰用户对文本的专注。用户可随时开始或暂停预览滚动，并对速度、样式等进行调节，调节结果会即时反映。整个交互过程力求直观，例如通过手势或少量控件即可完成大部分操作。

关键交互流程：
	•	进入预览模式：用户在编辑完文本和设置初步样式后，点击“预览滚动”按钮，应用进入全屏预览模式开始滚动显示文本。默认界面仅显示正在滚动的文字，其余控制按钮暂时隐藏（使界面简洁干净）。用户可以通过单击屏幕暂停/继续滚动（单击属自然交互，便于演讲者临时停顿），暂停状态下界面上浮现出控制面板半透明显示。
	•	**实时参数调节：**在预览滚动过程中，用户可呼出调节面板（例如轻触屏幕某一角落或使用手势上滑呼出）。调节面板以悬浮半透明方式叠加在屏幕下方，不遮挡主要文本区域 ￼。面板上提供关键参数的调控控件：速度滑块、字体大小调节、行距调节，以及颜色/亮度快捷按钮等。用户拖动滑块调整参数时，背景的文字滚动效果同步变化，方便用户观察并找到合适的参数值。调节完毕后用户可手动关闭面板或等待几秒无操作后面板自动隐退，以恢复无干扰的全屏提词界面。
	•	交互提示与引导：为保证交互的直观性，应用在首次使用预览模式时会进行简单的新手指引，例如提示用户“双指捏合”可以放大/缩小字体，上下滑动可以微调滚动速度等（这些手势作为按钮操作的自然辅助）。通过这些贴心的引导，用户可以快速掌握在预览界面下如何控制文本滚动，从而更加专注于演讲内容本身。

模块四：定时启动与远程/语音控制

模块说明：提供多样的控制方式，方便用户按照预定计划或通过自然手段控制滚动的开始、暂停和速度调整。本模块包含定时启动（倒计时）、语音控制以及远程控制三大功能，旨在让用户在各种场景下都能方便地掌控提词过程。这些控制方式可单独使用，也可结合应用：例如用户可以设置倒计时启动演讲，同时备用语音命令来暂停，或让助手通过另一设备协助控制。

关键交互流程：
	•	**定时启动（倒计时）：**用户在设置中启用“定时开始”功能，并设定一个倒计时时长（例如10秒）。当用户准备就绪后，点击“开始倒计时”，屏幕上显示一个醒目的倒计时读秒。在倒计时过程中，用户可以做好演讲姿态；倒计时结束时应用自动开始滚动文本，并按预设速度缓缓移动 ￼。这一功能确保演讲者无需手动按开始即可准时开讲，尤其适用于自行录像时留出准备时间。
	•	**语音控制播放/暂停：**应用通过集成系统语音识别或Siri快捷指令，实现对“开始”、“暂停”等口令的识别。当用户启用语音控制后，应用将在后台监听预设的简单语音命令（例如说出“开始”即相当于点击播放）。演讲者无需腾出手操作，只需短暂停顿发出指令即可控制提词器，非常自然便捷 ￼。例如，当需要临时停顿回答观众提问时，用户说“暂停”，滚动立即暂停；回答完毕说“继续”，滚动恢复。语音控制在嘈杂环境下可自动降灵敏度避免误触发，并提供语音反馈音效确认命令已执行。
	•	**远程控制（多设备协同）：**GlowBoard支持通过另一台设备远程操控提词滚动 ￼。典型场景是用户使用iPad作为提词屏幕，同时用iPhone充当遥控器。两设备上安装GlowBoard并通过蓝牙或局域网配对连接后，iPhone端会进入“远程控制器”界面：上面显示当前滚动文本的同步预览以及控制按钮。远程控制器可调整滚动速度、开始/暂停，以及快速跳转到下一段落等，所有操作都会实时反映在iPad主屏上 ￼。这样演讲者可将iPad放在远处视线可及的位置，由助手或自己通过手持的iPhone无线控制滚动进度。如果没有助手，演讲者自己也可以用隐藏在掌心的手机遥控，达到不接触提词屏幕即可控制的效果。此外，应用也支持简单的蓝牙外设控制（如蓝牙键盘的空格键暂停/播放，方向键调速等）供不同偏好的人使用 ￼ ￼。

模块五：用户体验优化与设置持久化

模块说明：该模块涵盖提升用户体验的辅助功能及应用状态的持久化保存。包括提供夜间模式、屏幕亮度控制、换页提醒等功能，确保不同环境下的使用舒适度和可读性；以及自动保存用户设置和状态，在应用重启后恢复上次会话，让用户无需重复设置。虽然这些不直接属于文本滚动核心，但对整体体验至关重要。

关键交互流程：
	•	**夜间模式切换：**用户可在应用主界面或控制中心快速切换夜间模式。一旦开启，应用界面的主色调变为深色，文本颜色自动调整为高对比度的浅色或用户设定的颜色，以适应弱光环境阅读。夜间模式的状态会被记录，并在下一次启动应用时自动延续，除非用户手动切换回来。
	•	**亮度调节：**在应用内提供屏幕亮度调节滑杆，用户可细化控制屏幕背光亮度，以适应当前环境需要（例如演播室强光下调高亮度，暗室中降低亮度防止刺眼）。应用调用系统接口调整设备屏幕亮度，或者通过覆盖半透明遮罩实现细粒度调暗效果。亮度设置同样会自动保存，下次打开时保持上次调定的亮度水平。
	•	换页/段提醒：当长稿件被划分为多个页面或当滚动即将到达预设的段落结束位置时，应用通过震动（设备支持时）或提示音提醒演讲者当前页即将结束。这一提示功能可由用户开启/关闭和设置提醒阈值（例如距离末行还有两行时提示）。实现时，在滚动文本到达阈值位置时触发设备短振动或播放轻提示音，从而给演讲者一个预备信号而不干扰观众。该设计借鉴了专业提词器使用中的经验，提高用户对长文本的掌控 ￼。
	•	**自动保存和恢复：**用户在应用中做出的所有设置调整（滚动速度、字体和颜色选择、夜间模式状态等）以及当前加载的文本和滚动位置，都会实时保存到本地存储。当用户退出应用或发生系统中断时，这些状态也会被记录。下次重新打开GlowBoard时，应用自动恢复到上次退出时的状态：包括打开同一篇文本，滚动停留位置、已设置的各项参数等。这使用户可以“续接”上次的工作，无须每次重新配置，从而大幅提升连贯性和易用性。
